# Oj Serializers

A faster JSON serializer for Ruby Objects. Easily migrate away from Active Model Serializers.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'oj_serializers'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install oj_serializers
